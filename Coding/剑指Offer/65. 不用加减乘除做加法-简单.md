## 题意

写一个函数，求两个整数之和，要求在函数体内不得使用 “+”、“-”、“\*”、“/” 四则运算符号。

---
## 题意

这道题目需要用二进制来做，我们可以发现，二进制的加法可以看作异或和与操作

比如 
1+0 0+1 进位为0 和为1 
1+1 0+0 进位为1 和为0

可以发现，**进位相当于与操作，和相当于异或操作**

```
c=a&b<<1 计算进位c
a=a^b 计算无进位和n，复制给a
b=c 将进位c赋值给b
```

```python
def add(a,b):
	x=0xffffffff
	a,b=a&x,b&x
	while b!=0:#直到没有进位，结束
		a,b=(a^b),(a&b)<<1&x
	return a if a<=0x7fffffff else ~(a^x)
```
