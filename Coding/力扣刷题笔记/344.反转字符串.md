## 题意

编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 char[] 的形式给出。

不要给另外的数组分配额外的空间，你必须原地修改输入数组、使用 O(1) 的额外空间解决这一问题。

你可以假设数组中的所有字符都是 ASCII 码表中的可打印字符。

**eg**

**输入：**\["h","e","l","l","o"\]
**输出：**\["o","l","l","e","h"\]

---
## 解题

本题比较简单，可以直接把数组倒序

```python
s[:]=s[::-1]
return s
```

或者使用双指针法，每次用两个指针指向剩余未交换的字符串的两头，交换完后向中间靠拢。

```python
l=len(s)
for i in range(l//2):
	s[i],s[l-i-1]=s[l-i-1],s[i]
return s
```
